Actividades realizadas durante el tercer periodo

Servicios y plataformas de Cloud Computing

Lo que se realizó fue una investigación bibliográfica de los principales conceptos sobre Cloud Computing y sus principales proveedores. Los conceptos básicos, los tipos de servicios y sobre los principales proveedores de servicios de la nube a nivel mundial. Los tres grandes proveedores (Google Cloud, Microsoft Azure y Amazon Web Services (AWS)).

Configuración del servicio SSH, servidor web Apache2 y despliegue de un sitio web con Ubuntu Server 22.04 LTS

Configuración básicas al servidor Ubuntu 22.04 LTS, configuración de una IP estática y la instalación del servidor SSH, para conexiones seguras de forma remota, además, poder instalar el servidor web Apache2 y desplegar un sitio html desde el sistema operativo de Linux.

Configuración de Servidores Web en Conjuntos de Disponibilidad con Microsoft Azure

En está actividad se realizaron diferentes configuraciones en Microsoft Azure, como crear un nuevo recurso, crear una puerta de enlace, crear máquinas virtuales, entre otros.

Desplegar un sitio web con servidor Nginx

La actividad consistía en realizar la instalación y configuración del servidor web Nginx en Ubuntu Server 22.04 LTS, para desplegar un sitio web personalizado el cual estaba en un repositorio de GitHub.
